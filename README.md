# Агент заточил циркулярный диск

На [ЗОКе](https://zok.cx/) [огнеборцев](https://zok.cx/game/3450/) прославляли. Вчера товарищ заделился фотками с производства. Сегодня я уже не выдержал.

Старый циркулярный диск совсем захворал: зубья затупились, местами отвалились кромки. Добил я его старыми оконными рамами и _внезапными_ гвоздями. Терять нечего, решил заострить.

Маркером пометил зуб и пошёл по кругу точить кромки. Заточились как попало, особо ни на что не рассчитывал. Прикрутил диск, пробнул — режет как по маслу.

Вывод: заточка рулит, гвозди — зло.

Спасибо, что вы на канале «Я у Лапки плотник» (а вы что, не знали?). Не переключайтесь.
